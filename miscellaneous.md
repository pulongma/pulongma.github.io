## Tools

- [Spacetime Explorer](https://pulongma.github.io/spacetime-explorer/): Interactive visualization of space-time covariance functions

## Python Package

- [gpbayeskit](https://github.com/pulongma/gpbayeskit): This package implements space-time models for spatio-temporal data (adapted from R packages GPBayes and STVGP)

## R Packages

- [STVGP](https://github.com/pulongma/STVGP): This package implements space-time covariance models with Vecchia GP approximation
- [GPBayes](https://cran.r-project.org/web/packages/GPBayes/index.html): This package serves a basic tool for Gaussian process modeling in uncertainty quantification
- [ARCokrig](https://cran.r-project.org/web/packages/ARCokrig/index.html): This package implements autoregressive cokriging for multifidelity codes (or computer models)
